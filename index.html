Lumens Tiny vanilla JavaScript carousel

Notice: This frontend is still undergoing a rework and is not finished yet, but Lumens can still be used as before.

Lumens is a lightweight JavaScript slideshow which works without any dependencies like JQuery or other bloated libraries. It doesn't rely on a own CSS so you have nothing messing with your personal styling.
Lumens was built for developers. It doesn't add any navigation and pagination but gives you all the tools to add them yourself. This way you have full control over what happens, and when it does so.

Compability

Since there are a brazillion possible combinations on devices, operating systems, browsers and versions, Lumens tries to be as compatible as possible with all of them. It uses very basic vanilla JavaScript even Internet Explorer 9 can handle. (Amazing, I know).

Lumens supports the most commen Browsers which include:

  • Google Chrome
  • Mozilla Firefox
  • Opera
  • Apple Safari
  • Microsoft Edge
  • Microsoft Internet Explorer (version 9 and above)
  • Samsung Internet

For a detailed list, you can check out this Sheet containing all the tests Lumens went through

Options

Like every slideshow ever in the existence of the internet, Lumens also comes with a bunch of options you can set (if you feel like it). To configure your experience

Following is a list of all the possible options you can set with their default values:

var defaultOptions = {
	slidesPerPage: 1, //How much slides are shown per page
	margin: 0, //The margin each slides has left and right in Pixels
	duration: 200, //Duration of the slide-animation in milliseconds
	easing: "ease-out", //The easing function used for transitions
	autoplay: false, //Duration between each slide in milliseconds
	draggable: true, //If false, the slides can no longer be dragged
	threshold: 20, //Minimum dragging-distance to change slide
	arrowControls: false, //Enable to allow controling slideshow by arrow-keys
	mouseButton: false, //Set to 0, 1 or 2 to define which mousebutton is needed to swipe
	preventClickDistance: 50, //Minimum dragging-distance to prevent clicks on slide
	responsive: [], //Object containing settings and breakpoints for responsive behaviour
	noOuterMargin: false, //Increase the width of the slideshow to compensate slide-margin
	startAtPage: 0, //The slide on which to start (Starts with 0)
	infinite: false //Set true to let the slideshow loop
}

Installation

Installing Lumens is as easy as it gets. Just download the compiled javascript here

Now import the script in the head of your HTML like always, and you'll be all set.

<script src="path/to/lumens.js"></script>

Now just generate a HTML-Container which contains a child for each slide you want.

Callbacks

Lumens provides you with callback functions for changing slides or changing the breakpoints during resizing.

Try swiping and resizing the window (below 1024px) to trigger callbacks.

var callbackslider = new Lumens(".callbackslider", {
	slidesPerPage: 2,
	responsive: [
	{
		breakpoint: 1024,
		settings: {
			slidesPerPage: 1
		}
	}
	]
})

callbackslider.changed(() => {
	//SLIDE CHANGED
})

callbackslider.resize(() => {
	//NEW BREAKPOINT REACHED
})
