Callbacks

Lumens provides you with callback functions for changing slides or changing the breakpoints during resizing.

Try swiping and resizing the window (below 1024px) to trigger callbacks.

Slide 1
Slide 2
Slide 3
Slide 4
Slide 5

 

var callbackslider = new Lumens(".callbackslider", {
	slidesPerPage: 2,
	responsive: [
	{
		breakpoint: 1024,
		settings: {
			slidesPerPage: 1
		}
	}
	]
})

callbackslider.changed(() => {
	//SLIDE CHANGED
})

callbackslider.resize(() => {
	//NEW BREAKPOINT REACHED
})

API

Use the given functions to control how the slideshow acts just as you want it to

Slide 2
Slide 3
Slide 4
Slide 5
apislider = new Lumens(".apislider", {
	draggable: false
})

document.getElementById("next").addEventListener("click", () => {
	apislider.gotoNext()
})

document.getElementById("prev").addEventListener("click", () => {
	apislider.gotoPrev()
})
